EmpowerHer AI Chatbot 💜
An AI-powered chatbot designed to support and empower women by providing guidance, resources, and helpful conversations.

Features
AI chatbot using Google Gemini API
Natural language conversations
User-friendly interface
Fast response generation
Technologies Used
Python
Streamlit
Google Gemini API
Installation

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py

Project Structure
empowerher-ai-chatbot/ │ ├── app.py ├── gemini.py ├── requirements.txt └── test.py
